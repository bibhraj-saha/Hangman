# Hangman

A modern, interactive Hangman game built with **Python** and **Streamlit**.

This project is a modernization of my original college Hangman game into a small, production-grade software engineering project suitable for my GitHub portfolio and resume.

---

## Project Goals

- Modern Python project structure
- Interactive Streamlit web interface
- Multiple word categories
- Difficulty levels
- Session state management
- Score tracking
- Restart and new game support
- Logging
- Configuration management
- Unit testing
- Executable packaging
- Professional documentation

The objective is **not** to build the largest Hangman game, but to demonstrate clean software engineering practices while keeping the project simple, maintainable, and executable.
