# Project Scope

## Project Title

Hangman — Modern Python & Streamlit Edition

---

# Background

This project is a modernization of an original college Hangman game.

The objective is to transform a basic console application into a clean, production-grade Python project that demonstrates modern software engineering practices while intentionally keeping the project small and maintainable.

---

# Problem Statement

Traditional Hangman implementations are often written as single-file console applications with limited structure and little emphasis on maintainability, testing, documentation, or user experience.

The goal of this project is to demonstrate how an existing educational project can be refactored into a professional portfolio application without significantly increasing its functional scope.

---

# Objectives

The project aims to:

- Improve project organization
- Modernize the architecture
- Introduce a graphical web interface
- Improve maintainability
- Demonstrate Python best practices
- Showcase clean software engineering

---

# Functional Scope

The application will provide:

- Interactive Hangman gameplay
- Random word selection
- Multiple categories
- Difficulty levels
- Score tracking
- Restart functionality
- Session management
- Instructions

---

# Non-Functional Scope

The project will include:

- Modular architecture
- Logging
- Configuration
- Unit testing
- Documentation
- Executable packaging
- Code formatting
- Static analysis

---

# Out of Scope

The following features are intentionally excluded:

- Multiplayer
- Online leaderboard
- User accounts
- Databases
- Cloud deployment
- REST APIs
- Docker
- Kubernetes
- Authentication
- Microservices
- AI gameplay

These exclusions help keep the project focused and appropriate for a small portfolio application.

---

# Target Audience

This project is intended for:

- Recruiters
- Hiring managers
- Python developers
- Software engineering students
- GitHub portfolio reviewers

---

# Success Criteria

The project will be considered complete when it:

- Runs as a Streamlit web application
- Follows modern Python project structure
- Includes automated testing
- Includes documentation
- Can be packaged as an executable
- Is suitable for inclusion in a professional portfolio

---

# Expected Outcome

A polished, maintainable, production-grade Hangman application that demonstrates practical software engineering skills while maintaining a deliberately small project scope.
